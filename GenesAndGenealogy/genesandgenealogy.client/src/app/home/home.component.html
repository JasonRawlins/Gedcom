Welcome to the Gedcom.NET Genes and Genealogy site. This is a demonstration of how Gedcom.NET might be used to create a custom site for displaying genealogical information from a gedcom file.

You can start exploring the site by viewing the profile of the fictional person Dylan Davis. His profile has the most complete collection of events, sources, and relationships. The Aberystwyth University is the most complete source.

Or you can view all people in the tree as a list.

The gedcom file used for this site was generated from a fictional tree on Ancestry.com named GedcomNetTestTree

You can view the code in the Gedcom repository on Github
